window properties and methods :

it is represent entire browser window

1. location.href :- This property represents url of the current web page running in the browser window.
2. navigator.userAgent :- This property represents the name of current browser.
3. screenX :- This property represents X(horizontally) value of current browser position on the screen.
4. screenY -This property represents Y(vertically) value of current browser position on the screen.
5. alert() :- This method displays an information dialogbox (message dialogbox) to the user.
6. confirm()
7. print() :-This method displays print dialogbox, which is used to print the current webpage through selected printer.
8. setTimeout() :-• This method calls the specified function, after completion of specified no. of milli seconds. • Note: 1000 milli seconds = 1 second Syntax: window.setTimeout(function, milli seconds
9. setInterval()
10. open()
11. clearInterval
